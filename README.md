# 📬 Mailbox a React Firebase Mail App

A responsive email client application built with **React** and **Firebase**, supporting real-time email sending, receiving, and authentication using Firebase Authentication and Realtime Database.

## 🚀 Features

- ✅ User Authentication (Signup/Login)
- ✉️ Compose and send emails to other registered users
- 📥 View received emails in the Inbox
- 📤 Access sent mails in the Sent section
- 🔄 Real-time updates using Firebase Realtime Database
- ⚙️ Clean UI using React-Bootstrap
- 🧠 LocalStorage-based user session management
- ⚠️ Recipient validation before sending

## 🛠 Tech Stack

- **Frontend**: React (Vite), React Bootstrap, React Router
- **Backend-as-a-Service**: Firebase (Auth + Realtime Database)
- **Other Tools**: React Toastify (notifications)
